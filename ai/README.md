# AI Chatbot Frontend

A production-quality AI chatbot interface built with Next.js that connects to a LangChain Agent backend endpoint.

## Features

- 🚀 **Streaming Responses** - Real-time streaming of AI responses
- 💬 **Modern Chat UI** - Clean, responsive design with message bubbles
- 📝 **Markdown Support** - Renders markdown with syntax highlighting for code blocks
- 📋 **Copy to Clipboard** - Easy copying of messages and code snippets
- 🛑 **Abort Generation** - Stop AI response generation mid-stream
- ⌨️ **Keyboard Shortcuts** - Enter to send, Shift+Enter for new line
- 🔄 **Loading States** - Visual indicators for loading and streaming
- ❌ **Error Handling** - Graceful error handling with retry options
- 📱 **Mobile Responsive** - Works on all device sizes

## Tech Stack

- [Next.js 14](https://nextjs.org/) - React framework
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [React Markdown](https://github.com/remarkjs/react-markdown) - Markdown rendering
- [React Syntax Highlighter](https://github.com/react-syntax-highlighter/react-syntax-highlighter) - Code block formatting
